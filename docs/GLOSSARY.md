# Glossary

This file is an optional terminology reference.

- `slot`: trip instance
- `slotUid`: stable trip identifier used by selling flows
- `manual slot`: trip created manually
- `generated slot`: trip created from a template
- `cutoff`: point after which a role may no longer sell for a trip
- `presale`: ticket sale record
- `seller`: selling role
- `dispatcher`: dispatch role
- `season_id`: season identifier in `YYYY` form
- `business_day`: business date in `YYYY-MM-DD` form
